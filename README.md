# Retail Sales Analytics

This project analyzes an e-commerce transaction dataset to extract business insights.

## Dataset

The dataset used is the Online Retail dataset.

Download it from:
https://www.kaggle.com/datasets/carrie1/ecommerce-data

Save the file as:

online_retail.csv

## Analysis Performed

The notebook performs several analytics tasks:

• Data cleaning and preprocessing  
• Revenue calculation  
• Revenue by country  
• Top customers by revenue  
• Monthly revenue trend analysis  

## Tools Used

Python  
pandas  
matplotlib  

## Notebook

The full analysis is available in:

retail_analytics.ipynb
